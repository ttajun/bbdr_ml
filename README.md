# bbdr_ml

bbdr_ml is python exectuable command for ML.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install bbdr_ml

```bash
pip install bbdr-ml
```

## Usage

```console
# create ML model
[user@dev]$ bbdr_mmaker -h

# Predict
[user@dev]$ bbdr_predict -h
```
